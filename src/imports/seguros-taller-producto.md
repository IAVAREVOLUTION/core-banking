Subcategoría “Seguros” en el Taller de Producto — Prompt Mejorado
Se debe crear una nueva Subcategoría en el Taller de Producto llamada “Seguros”, con las siguientes reglas y ajustes obligatorios:

Configuración general
La Línea de Producto debe tener como valor default “Seguros”.

El campo “Tipo Tasa” debe cambiar su etiqueta a “Tipo Cobertura”.

Subpestañas requeridas
La Subcategoría “Seguros” debe incluir las siguientes subpestañas, con el comportamiento indicado:

Default

Periodos

Matriz de Tasas → renombrar a “Montos y Coberturas”

Requisitos

Paquetes

Debe aplicar el nuevo enfoque:

Mostrar únicamente productos cuya Línea de Producto = “Seguros”

Sucursales

Cargos

Prelación de Cargo

Fases

Garantías

Impuestos

Comisiones

Expediente Electrónico

Validar consistencia con la subpestaña Requisitos

Autorizaciones

Actualmente vacío; se debe definir qué configuraciones requiere

Eventos Contables

Actualmente vacío; se debe definir qué configuraciones requiere

Modal de Paquetes (Jerarquía de Productos)
Cuando se agregue un producto relacionado (Paquete):

Debe abrirse un modal que liste productos donde:

Línea de Producto = Crédito o

Línea de Producto = Seguros

Los campos visibles en el modal deben ser:

Línea de Producto

Tipo (sinónimo de Sub‑línea)

Nombre del Producto

Estos mismos campos deben mostrarse en la subpestaña Paquetes.
